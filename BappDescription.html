This extension unescapes unicode-escaped sequences in HTTP requests and responses, as well as WebSocket messages. It helps uncover obfuscated payloads or identifiers that use multiple layers of Unicode escaping, improving visibility for manual inspection.

Features

  • Automatically decodes nested Unicode sequences in HTTP messages
  • Adds a custom tab for viewing the unescaped content of selected requests and responses
  • Provides "pretty print" functionality for JSON content

Usage

  1. Select a request or response in the HTTP history or other Burp tool.
  2. Navigate to the "UnUnicode" tab to view the decoded version of the message.
  3. Use the decoded output to better understand any obfuscated content.
